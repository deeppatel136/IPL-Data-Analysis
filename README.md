# IPL Data Analysis 🏏

## 📌 Project Overview
This project analyzes IPL (Indian Premier League) data using Python to extract insights about teams and players.

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib

## 📊 Analysis Performed
- Top Teams by Total Runs
- Top Batsmen in IPL
- Toss Impact on Match Result

## 🔍 Key Insights
- Mumbai Indians scored highest total runs
- Virat Kohli is the top run scorer
- Toss has moderate impact on match outcome

## 📁 Project Structure
- IPL_Analysis.ipynb → Main analysis notebook

## 🚀 How to Run
1. Download the notebook
2. Open in Jupyter Notebook
3. Run all cells
